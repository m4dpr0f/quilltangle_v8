The player begins as pure potentialâ€”a **Chaos Shard**, associated with D12 Ether.

- No Garu companion yet
- Player exists as ethereal consciousness
- Can observe the Quillverse but limited interaction
- Must complete TEK8 quiz to determine elemental affinity
- Shard color shifts based on elemental leanings

### Shard Activities
- Take TEK8 assessment

- **3 days neglect**: Egg enters "cold" state, needs warming
- **7 days neglect**: Egg goes dormant, 3-day recovery needed
- **14 days neglect**: Egg "sleeps" indefinitely until 7-day recovery
- **Eggs cannot die from neglect** - they just pause development

This is intentionally forgiving to match wellness philosophy: no punishment, just natural consequences that can be recovered from.

### Egg Spawning Formula

The number and distribution of eggs spawned depends on:

```
BASE EGGS = 1 + (Garu Level / 10)

BONUS EGGS per territory type:
  + 1 egg per unique Road visited
  + 2 eggs per Road where Garu won a race
  + 3 eggs on the Road where Garu was born
  + 1 egg per Nation where Garu was welcomed

ELEMENT DISTRIBUTION:
  - Eggs inherit parent's elemental mixture
  - Higher-level elements = more eggs of that type
  - Rainbow Bridge Garu (all elements 50+) = eggs of ALL types

PLACEMENT:
  - One egg spawns on EVERY Octix the Garu touched in life
  - Additional eggs cluster near places of significance
  - Home territory gets the most eggs
```

## Database Schema

```sql
-- Core Garu table
CREATE TABLE garu (
  id SERIAL PRIMARY KEY,
  owner_wallet TEXT NOT NULL,
  name VARCHAR(50) NOT NULL,
  primary_element VARCHAR(20) NOT NULL,
  secondary_element VARCHAR(20),
  composite_type VARCHAR(50), -- NULL for basic, "Magma", "Rainbow Bridge", etc.

  -- Stats (0-100 each)
  stat_fire INTEGER DEFAULT 10,
  stat_earth INTEGER DEFAULT 10,
  stat_air INTEGER DEFAULT 10,
  stat_water INTEGER DEFAULT 10,
  stat_ether INTEGER DEFAULT 10,
  stat_chaos INTEGER DEFAULT 10,
  stat_order INTEGER DEFAULT 10,
  stat_coin INTEGER DEFAULT 10,

  level INTEGER DEFAULT 1,
  experience INTEGER DEFAULT 0,
  rider_bond INTEGER DEFAULT 50, -- 0-100

  -- Lifecycle
  phase VARCHAR(20) DEFAULT 'egg', -- shard, egg, hatched, fused, dead
  hatched_at TIMESTAMP,
  died_at TIMESTAMP,
  death_location TEXT, -- road/octix where died

  -- Lineage
  parent_garu_ids INTEGER[], -- for fusion tracking
  generation INTEGER DEFAULT 1, -- increases with each fusion
  birth_territory TEXT, -- D8 wilderness location

  -- Metadata
  image_url TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Territory visits (for egg spawning on death)
CREATE TABLE garu_territory_visits (
  id SERIAL PRIMARY KEY,
  garu_id INTEGER REFERENCES garu(id) ON DELETE CASCADE,
  territory_id TEXT NOT NULL, -- road ID like D4OUT
  visit_type VARCHAR(20), -- visited, raced, won, lived
  first_visited TIMESTAMP DEFAULT NOW(),
  visit_count INTEGER DEFAULT 1,
  UNIQUE(garu_id, territory_id)
);

-- Eggs waiting to be claimed
CREATE TABLE garu_eggs_wild (
  id SERIAL PRIMARY KEY,
  territory_id TEXT NOT NULL,
  parent_garu_id INTEGER, -- NULL for primordial eggs
  elements TEXT[] NOT NULL, -- inherited from parent or random for primordial
  potency INTEGER DEFAULT 1, -- based on parent level/generation
  spawned_at TIMESTAMP DEFAULT NOW(),
  claimed_by TEXT, -- wallet that claimed it
  claimed_at TIMESTAMP,
  expires_at TIMESTAMP -- eggs decay after 30 days unclaimed
);

-- Daily wellness care logs
CREATE TABLE garu_care_logs (
  id SERIAL PRIMARY KEY,
  garu_id INTEGER REFERENCES garu(id) ON DELETE CASCADE,
  user_wallet TEXT NOT NULL,
  care_date DATE DEFAULT CURRENT_DATE,

  -- 8 dimensions (boolean: did they care today?)
  physical BOOLEAN DEFAULT false,
  emotional BOOLEAN DEFAULT false,
  intellectual BOOLEAN DEFAULT false,
  social BOOLEAN DEFAULT false,
  occupational BOOLEAN DEFAULT false,
  spiritual BOOLEAN DEFAULT false,
  environmental BOOLEAN DEFAULT false,
  financial BOOLEAN DEFAULT false,

  -- Details
  physical_notes TEXT,
  emotional_notes TEXT,
  intellectual_notes TEXT,
  social_notes TEXT,
  occupational_notes TEXT,
  spiritual_notes TEXT,
  environmental_notes TEXT,
  financial_notes TEXT,

  -- Instrument practice
  instrument_practiced BOOLEAN DEFAULT false,
  instrument_minutes INTEGER DEFAULT 0,

  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(garu_id, care_date)
);

-- Garu fusion history
CREATE TABLE garu_fusions (
  id SERIAL PRIMARY KEY,
  parent1_id INTEGER NOT NULL,
  parent2_id INTEGER NOT NULL,
  child_id INTEGER REFERENCES garu(id),
  fusion_shrine_location TEXT,
  fused_at TIMESTAMP DEFAULT NOW(),
  rider1_wallet TEXT NOT NULL,
  rider2_wallet TEXT NOT NULL
);

-- Indexes
CREATE INDEX idx_garu_owner ON garu(owner_wallet);
CREATE INDEX idx_garu_phase ON garu(phase);
CREATE INDEX idx_wild_eggs_territory ON garu_eggs_wild(territory_id) WHERE claimed_at IS NULL;
CREATE INDEX idx_care_logs_date ON garu_care_logs(garu_id, care_date);
```
