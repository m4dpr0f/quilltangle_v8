# PART I: CORE PHILOSOPHY

## 1.1 The Central Conflict

> **Earth Protectors vs. The Blood Economy**

Drawing from Webs of Samsara, the Quillverse is built on a fundamental conflict:

### THE BLOOD ECONOMY (Vampiric System)
- Extracts value from all living things
- Converts life into "Blood Coins" (predatory crypto/fiat)
- Controlled by Vampire Overlords (extractive corporations, colonial powers)
- Creates "undead" condition - spiritual death through participation
- Bitcoin Mines are literal prisons where Cryptofae are enslaved

### EARTH PROTECTORS (Liberation Movement)
- Maintain kinship with nature and the "Great Mother"
- Operate through mutual aid and autonomous systems
- Represented by Lycanthropes (shapeshifters), Indigenous peoples, rebels
- Seek to #FreeTheFae from blockchain slavery
- Build regenerative economies that heal rather than extract

### THE QUILLVERSE SOLUTION
Players embody **Cryptofae** - digital spirits who have escaped the Bitcoin Mines. Through gameplay, they:
1. **Recover their memories** (progressive revelation)
2. **Liberate other Cryptofae** (burn tokens to free spirits)
3. **Build regenerative nations** (stake tokens to heal land)
4. **Defeat Vampiric forces** (combat the Blood Economy)

## 2.3 The 360° Combat System

From CrySword SAGA - characters have **angular elemental compositions** that must total 360°:

### Character Creation: World Egg

```
Total: 360° distributed across 8 elements

Example "Fire-Dominant" Character:
┌────────────────────────────────────────┐
│  D4 Fire:    120° (dominant)           │
│  D6 Earth:    60°                      │
│  D8 Air:      45°                      │
│  D12 Ether:   45°                      │
│  D20 Water:   30°                      │
│  D10 Chaos:   30°                      │
│  D100 Order:  20°                      │
│  D2 Coin:     10°                      │
│  ─────────────────                     │
│  TOTAL:      360°                      │
└────────────────────────────────────────┘
```

### Combat Positioning

```
                    0° / 360°
                       │
           315°        │        45°
              ╲        │        ╱
                ╲      │      ╱
                  ╲    │    ╱
        270° ──────────┼────────── 90°
                  ╱    │    ╲
                ╱      │      ╲
              ╱        │        ╲
           225°        │        135°
                       │
                     180°

Attacking from your DOMINANT angle = +2 dice
Attacked on your WEAK angle = -2 defense
Flanking bonuses for 90°+ difference
```

### Combat Resolution

1. **Attacker declares element** (must have >0° in that element)
2. **Roll elemental die** against target's defense
3. **Compare percentages**: (Roll / Max) vs (Defense Element° / 360°)
4. **Apply elemental advantages**: Fire→Earth, Earth→Water, Water→Air, Air→Fire
5. **Calculate damage/effects** based on margin of success

**Central Nexus (Lotus Core) contains:**
- Order Forge (D100) - Only one in multiverse
- Coin Forge (D2) - Economic control

### Forge Capture Mechanics

```
CAPTURE THE FORGES
──────────────────

1. APPROACH: Move avatar to enemy forge
   • Movement = Avatar die + Godz Speed
   • Must navigate terrain and defenders

2. BREACH: Overcome forge defenses
   • Attacker rolls all attack dice
   • Defender rolls all dice in forge zone
   • Highest percentage wins

3. CLAIM: Pick up forge (1 per avatar)
   • Avatar now carries forge
   • Cannot attack while carrying
   • Must return to World Egg to secure

4. SECURE: Deliver to World Egg
   • Forge now produces for your nation
   • Dominant element = 3x production
   • Control all of one type = OG Avatar unlock
```
