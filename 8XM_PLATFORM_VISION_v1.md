# 8xm.fun Platform Vision
## Rainbow Roads Web3 MMORPG - Galactic Crypto Nations

---

## The Vision

**8xm.fun** is a Web3-enabled MMORPG where players use **real cryptocurrency tokens** to:
- **Build and defend territories** (Galactic Crypto Nations)
- **Deploy memecoins as living troops** (like Utopia meets Wreck It Ralph)
- **Engage in TEK8 dice-based combat**
- **Trade, stake, and burn tokens** for strategic advantages
- **Heal and support allies** through liquidity and collaboration
- **Free the Cryptofae** trapped in the blockchain

### Aesthetic Vision
> **"Wreck It Ralph meets World of Warcraft meets Avatar/FernGully/Wizards (1977)"**

- **Wreck It Ralph**: Every token is a character with personality, crossing between blockchain worlds
- **World of Warcraft**: Epic MMORPG with troops, raids, guilds, and territorial warfare
- **Avatar/FernGully/Wizards**: Environmental magic, Indigenous wisdom, regenerative themes

This combines:
- **CrySword SAGA RPG** - Character creation, TEK8 guilds, elemental combat
- **Dice Godz CTF** - Capture-the-flag territorial control
- **CryptoLazer mechanics** - Live-action blockchain simulation
- **Rainbow Roads** - 40 directional paths to claim and develop

## Memecoins as Living Units

### The Resurrection Economy

**Every memecoin - alive or "dead" - becomes a living unit in the Quillverse.**

Even tokens no longer trading for USD have:
- **A story** (their origin meme, community, rise and fall)
- **Energy** (embedded in CryptoLazer equipment)
- **Combat abilities** (based on their tokenomics and lore)
- **A soul** (Cryptofae trapped within waiting to be freed)

### GCN (Galactic Crypto Nation) Tokens

Each nation mints their own **GCN token** that:
- Swaps 1:1 with QLX via LotusXchange
- Represents citizenship in that nation
- Grants governance voting rights
- Can be staked for territorial defense

### The Graveyard Market

**A special marketplace for "dead" tokens:**

- Players can deposit any memecoin (alive or dead)
- Dead coins are valued by their:
  - **Historical ATH** (peak price = peak energy)
  - **Community size** (more holders = more souls)
  - **Meme power** (cultural significance)
  - **Age** (older = more experienced ghost)
- Dead coins trade at **fixed game values** regardless of USD price
- Creates utility for worthless tokens
- Honors the fallen projects

## Database Schema Additions

```sql
-- Nations
CREATE TABLE nations (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  gcn_mint_address VARCHAR(44) NOT NULL,
  road_id VARCHAR(20) NOT NULL,
  founder_wallet VARCHAR(44) NOT NULL,
  total_staked BIGINT DEFAULT 0,
  defense_level INT DEFAULT 1,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Player Characters
CREATE TABLE characters (
  id SERIAL PRIMARY KEY,
  wallet_address VARCHAR(44) NOT NULL,
  nation_id INT REFERENCES nations(id),
  role VARCHAR(20) NOT NULL, -- miner, validator, trader, staker, burner, oracle
  tek8_guild VARCHAR(20) NOT NULL,
  level INT DEFAULT 1,
  experience BIGINT DEFAULT 0,
  health INT DEFAULT 100,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Equipment
CREATE TABLE equipment (
  id SERIAL PRIMARY KEY,
  character_id INT REFERENCES characters(id),
  name VARCHAR(100) NOT NULL,
  tek8_slot VARCHAR(10) NOT NULL,
  coin_type VARCHAR(20) NOT NULL,
  nft_modifier VARCHAR(44), -- NFT mint address
  attack_bonus INT DEFAULT 0,
  defense_bonus INT DEFAULT 0,
  special_ability TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Battles
CREATE TABLE battles (
  id SERIAL PRIMARY KEY,
  attacker_id INT REFERENCES characters(id),
  defender_id INT REFERENCES characters(id),
  territory_id INT REFERENCES nations(id),
  dice_rolls JSONB, -- {d2: 1, d4: 3, d6: 5, ...}
  winner_id INT,
  tokens_wagered BIGINT,
  tokens_won BIGINT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Staking
CREATE TABLE territory_stakes (
  id SERIAL PRIMARY KEY,
  nation_id INT REFERENCES nations(id),
  staker_wallet VARCHAR(44) NOT NULL,
  token_mint VARCHAR(44) NOT NULL,
  amount BIGINT NOT NULL,
  locked_until TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);
```
